# Clones Project 🚀

A curated collection of pixel-perfect website clones built to demonstrate advanced frontend development skills, UI/UX replication, and responsive design principles.

## 📁 Projects Included

### 1. Netflix Clone 📺
A faithful recreation of the Netflix streaming platform interface.
- **Features**: Responsive landing page, professional sign-in page, and dark-themed glassmorphism.
- **Tech**: HTML5, CSS3.
- **Pages**: 
  - `netflix_clone.html` (Landing Page)
  - `netlifx_sign_in_clone.html` (Sign-in Page)

### 2. Nike Clone 👟
A comprehensive e-commerce interface inspired by Nike's modern web design.
- **Features**: Multi-category navigation (Men, Women, Kids), product showcases, and interactive hover effects.
- **Tech**: HTML5, CSS3, Flexbox/Grid.
- **Sections**:
  - `index.html` (Hero/Home)
  - `men.html`, `women.html` (Category Pages)
  - `shoes.html`, `athelepicks.html` (Specialty Pages)

---

## 🛠️ Core Technologies
- **HTML5**: Semantic layout for accessibility.
- **CSS3**: Custom styling, animations, and responsive layouts (no external frameworks used, showcasing pure CSS mastery).

## 🚀 Getting Started
To view any of the clones, simply navigate to the respective folder and open the main HTML file in your web browser.
